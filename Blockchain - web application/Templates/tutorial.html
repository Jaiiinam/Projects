Tutorial



Step 1: Create wallets


In this step we look at how to create wallets.

Go to the Create wallet page and create a wallet. Make sure to store the wallet information.

Each wallet will contain information such as an Account address (public key), a Key to the account (private key - must be kept secret, will be used to transfer funds), and a balance of 10 coins (default).

Go to the Balances page, you will see all wallets and their balances.


Step 2: Create Transactions


In this step we look at how to transfer funds between wallets.

Go to the Transactions page and transfer funds between wallets. Make sure to create more than 10 transactions.

Once you have create 10 transactions please visit Mempool page this will show you all transactions.

Every time you mine a block those 10 transactions will be selected to create a merkle root.


Step 3: Mine Block


Go to Mine Block and click on the mine block.page

You will receive recently mined block with header, transactions, and a block hash.

Normally to mine a block you will be asked to submit a solution to a puzzle. However for simplicity sake we are just allowing you to mine blocks without any kind of data.


Step 4: Check Blockchain


In this step we will verify the blockchain. Please visit Check Blockchain page.

In this verification we are checking each block and block's hash, it's previous hash, and it's merkle root.

If result is not "ok" then blockchain was tempered or hacked.
